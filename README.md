# تطبيق محروقات

تطبيق بسيط لإدارة بسطة محروقات: مازوت، بنزين، غاز.

## التعديلات في هذه النسخة
- اسم التطبيق: **محروقات**.
- أيقونة مخصصة للمحروقات.
- تفعيل أولي برمز المالك عند أول تشغيل على أي جهاز جديد.
- لا تظهر رموز الدخول الابتدائية داخل التطبيق.
- يوجد مستخدمان: مدير وموظف.

## صلاحيات المدير
- تعيين القيم الابتدائية.
- تعديل المخزون والأسعار.
- شراء الموارد.
- البيع نقدًا أو بالدين.
- تسجيل الديون لنا وعلينا.
- قبض ديون الزبائن ودفع ديون الموردين.
- حذف الديون.
- مشاهدة السجل الكامل والفلترة حسب الاسم.

## صلاحيات الموظف
- البيع فقط، نقدًا أو بالدين.
- اختيار العملة: دولار / ليرة تركية / ليرة سورية.
- إدخال كمية البيع فقط، والسعر يُحسب من سعر المدير.
- رؤية المخزون وأسعار البيع.
- تسجيل دين للزبون.
- قبض / توفية دين الزبون.
- مشاهدة السجل والفلترة حسب اسم الزبون أو المورد.
- لا يستطيع الشراء، ولا تعديل السعر، ولا تعديل القيم، ولا حذف الديون، ولا دفع ديون الموردين.
